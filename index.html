(P)WA - (Progressive) Web Application

How to use fetch api in javascript to retrieve data from a RESTapi endpoint. The response renders using HTML markup coupled with the Bootstrap 4 CSS framework for the styling part - no extras CSS needed!

Note that the codebase is incomplete to make this page a true PWA. This demo page intends to only showcase how to fetch data from RESTApi endpoints using fetch api in Javascript.

Scotch.io has a very handy online guide so have a read and come back with a Pull Request and let's turn this Web App into a Progressive Web App!
